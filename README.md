# moneyfasting
Aplikasi MoneyFasting merupakan aplikasi sederhana yang digunakan untuk mencatat wishlist beserta harga barang, kemudian membandingkannya dengan jumlah tabungan yang dimiliki. Aplikasi ini membantu pengguna mengetahui apakah saldo mereka sudah mencukupi untuk membeli barang tertentu melalui proses pengecekan otomatis.
